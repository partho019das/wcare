# WeCare

WeCare is a modern healthcare appointment booking platform where patients can discover doctors, view detailed profiles, and book appointments online with a seamless user experience.

LIVE_SITE_URL_HERE

## Features

* Browse and explore available doctors with detailed information.
* View doctor profiles including specialty, experience, fee, and availability.
* Book appointments online with a simple and user-friendly form.
* Secure authentication and protected dashboard using JWT.
* Manage appointments with update and delete functionality.
* Personalized user dashboard with booking history.
* User profile management with editable information.
* Responsive design optimized for mobile, tablet, and desktop devices.

## Technologies Used

* Next.js
* React.js
* Tailwind CSS
* Node.js
* Express.js
* MongoDB
* JWT Authentication

## Getting Started

```bash
npm install
npm run dev
```

Open `http://localhost:3000` in your browser to view the application.
